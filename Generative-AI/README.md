# Generative AI Chatbot Project

This project is a web-based generative AI chatbot interface built using HTML, CSS, and JavaScript, with integration of Gradio for the AI functionality.

## Features

- Interactive chatbot interface
- Customizable profile section
- Social media sharing buttons
- Tooltips for various AI tools and platforms
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Gradio (for AI model integration)

## How to Run

1. Clone the repository
2. Open `genaichatbot.html` in a web browser
3. The Gradio app should load automatically

## Project Structure

- `genaichatbot.html`: Main HTML file
- `genaichatbot.css`: Stylesheet
- `genaichatbot.js`: JavaScript for interactivity

## Setup Instructions for AI Services

If you need to set up the backend AI services, here are guides for various platforms:

LangChain is a framework for building applications with LLMs. Visit https://www.langchain.com/

### Hugging Face
Platform for machine learning models: https://huggingface.co/

### Gradio
